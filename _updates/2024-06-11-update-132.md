# What is new?

## Ledger Device support

The last few months, the Tari community has been working on Ledger integration. Unfortunately, Tari won't be included in Ledger Live any time soon, mainly due to the architecture of Ledger Live, but there should soon be support for using Ledger hardware devices with Tari's console wallet.

## View Keys
[PR #6353](https://github.com/tari-project/tari/pull/6353) enables view keys on Tari addresses. The address is unfortunately longer, but having view keys allows for more scenarios and potentially lite wallets in the future. The next reset of Nextnet will include this upgrade.

## Layer Two Devnet
The layer two (Ootle) devnet is up and running. This network is mainly targeted at contract developers, so will have a fixed number of validator nodes running to ensure stability. You can use the indexer [https://indexer-devnet.tari.com](https://indexer-devnet.tari.com) to connect your asset vault. There is also a test Metamask Snap available. You can try it out at [https://splendorous-churros-36e6bf.netlify.app/]. 

## Monerokon Workshop
Tari contributor @sdbondi created a workshop for Monerokon 2024. Here is a link to the workshop if you want to try it out: [https://github.com/sdbondi/monerokon]

# What is next?

## Launchpad stability fixes
There have been a number of issues that users have experienced with Launchpad. The next version of Launchpad is nearly ready and will hopefully aliviate these problems. More on that soon.
